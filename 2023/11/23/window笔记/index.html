解决**0x80190001 错误**

对于错误代码 0x80190001 的解决方法

尝试一

可以尝试以下步骤：

首先，按下 Win 键+R 打开“运行”对话框，然后输入 inetcpl.cpl 并按下回车键。

在弹出的 Internet 属性框中，点击“高级”，然后找到 TLS 相关的选项，将它们全部勾选，然后点击“应用”。

重启你的电脑。

根据报告，这时候应用商店和浏览器都应该能够正常运行了

尝试二

如果系统是Windows10/11出现这样错误，初步判断可能是Windows 的网络连接引起的问题，或者是系统当前网络的问题，也不完全排除系统 bug。

如出现账户无法登录，商店无法访问，个人账户无法验证同步等问题，

试下以下修复方案：

提示：如果当前系统正在用梯子等软件请先关闭！

先尝试用输入命令来修复，步骤如下：

1.打开 Windows 搜索（快捷键 win+s）框输入CMD，会出现命令提示符，右键以管理员身份运行；

2.在命令框内输入netsh int ip reset回车，接着再输入命令netsh winsock reset回车来重置网络，最后重启电脑即可，部分电脑可以恢复，如果还是出现了0x80190001错误提示继续以下步骤。

尝试三

步骤 1：右键网络，点击属性，点击所连接的网络，并点击属性

步骤 2：打开后，勾选Internet 协议版本 4 (TCP/IPv4)**并双击，勾选使用下面的 DNS 服务器地址**， 如 114.114.114.114、8.8.8.8、8.8.4.4

步骤 3：返回到网络和共享中心，点击左下角Internet 选项，选择高级，点击重置（勾选删除个人设置）并确认重置即可

最后，重启电脑0x80190001错误就解决啦！
