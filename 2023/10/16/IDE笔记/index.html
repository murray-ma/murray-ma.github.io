快捷键

一、查找相关

Shift + Shift 在项目所有目录中查找

Ctrl + F 当前文件查找文本

Ctrl + Shift + F 当前项目查找文本

Ctrl + N 查找类

Ctrl + Shift + N 查找文件

Ctrl + Shift + Alt + N 查找类中的方法或变量

Ctrl + R 当前文件替换文本

Ctrl + Shift + R 当前项目替换文本

Ctrl + E 最近打开的文件

Ctrl + Shift + E 最近更改的文件

Alt + Shift + C 对比最近修改的代码、查找修改的文件

Alt + F7 定位函数或者变量或者类的所有引用到的地方

Ctrl + H 显示类结构图

Ctrl + Q 显示注释文档、查看 JavaDoc

Alt ＋ Q 看到当前方法的声明

Ctrl + P 参数信息（在方法中调用参数）

Ctrl + Alt + F12 资源管理器打开文件夹

Ctrl + F12 显示当前文件的结构

Shift + F1 外部文档

Ctrl + F1 显示错误描述或警告信息

Alt ＋ F1 将正在编辑的元素在各个面板中定位、查找文件所在目录位置

二、编辑相关

Alt + Shift + C 查看最近的变化

Ctrl + Shift + Enter 语句完成

Ctrl + Alt + Enter 向上插一行

Shift + Enter 向下插一行

Shift + F6 重构-重命名：重命名类、方法、变量等

Ctrl + F6 更改签名

Ctrl + R 当前文件替换文本

Ctrl + Shift + R 当前项目替换文本

Ctrl + D 复制当前行到下一行

Ctrl + Shift + V 复制多个文本

Ctrl + X 删除、剪切当前行

Ctrl + Y，删除当前行

Ctrl + Z 撤销

Ctrl + Shift + Z 取消撤销

Ctrl ＋ Shift ＋ Backspace 跳转到上次编辑的地方

Ctrl + Alt + O 优化导入的类和包

Ctrl ＋ O 选择父类的方法进行重写

Ctrl + I 实现方法

Alt + Insert 生成代码(如 get,set 方法,构造函数等)

Alt + Enter，导入包，自动修正

Ctrl ＋ Shift ＋ Insert 选择剪贴板内容并插入

Alt + Shift ＋ Insert 切换光标选择模式（选择内容多个光标）

Ctrl + Alt + T 把代码包在一个块内，例如：try/catch

Ctrl + Alt + I 自动缩进

Ctrl + Alt + L 自动格式化代码

Ctrl + G 定位行

Ctrl + Alt + V 引入变量

Ctrl + 空格 代码提示(被输入法占用)

Ctrl + Shift + Space 自动补全代码

Ctrl + Alt + Space 类名或接口名提示

Ctrl + Shift + Up/Down 在类或方法内部上/下移一行

Ctrl ＋ Alt ＋ Up/Down 快速跳转搜索结果

Ctrl + Up/Down 光标跳转到第一行或最后一行下

Ctrl + Alt + Left/Right 后退、前进（返回上次编辑的位置 ）

Alt + Shift + Up/Down 上/下移一行

Alt + Up/Down 在方法间快速移动定位

Alt + Left/Right 切换代码视图

Ctrl + Alt + N 内联

Ctrl + Alt + M 提取方法

Ctrl + Alt + V 提取属性

Ctrl + Alt + F 提取字段

Ctrl + Alt + C 提取常量

Ctrl + Alt + P 提取参数

Ctrl + W 选中代码，连续按选中不同代码块

Ctrl+Shift+F7 选中文本高亮显示（全部），按 Esc 高亮消失

Ctrl ＋ F7 查询当前元素在当前文件中的引用，然后按 F3 可以选择

Alt + F7 定位函数或者变量或者类的所有引用到的地方

F2 或 Shift + F2 高亮错误或警告快速定位

F3 向下查找关键字出现位置

Shift + F3 向上一个关键字出现位置

Alt+F3 选中文本逐个高亮显示，按 Esc 高亮消失

Ctrl + J 自动代码（eg：serr）

Ctrl + Shift + U 大小写转化

Ctrl + U 跳转到父方法、父类

Ctrl + P 方法参数提示

Ctrl + / 单行注释

Ctrl + Shift + / 多行注释

Ctrl + ] / [ 跳转到代码块结束、开始

Ctrl + Shift + ] / [ 选定代码块结束、开始

Ctrl + ”+ / -” 当前方法展开、折叠

Ctrl + Shift + ”+ / -” 全部展开、折叠

Ctrl + B 快速打开光标处的类或方法、找变量的来源

Ctrl ＋ Alt ＋ B 跳转到抽象方法的实现、找所有的子类

Ctrl + Shift + B 找变量的类

Ctrl + Shift + C 复制路径

Ctrl + Alt + Shift + C 复制引用，必须选择类名

Shift ＋ Click 关闭文件

Ctrl + F4 关闭

Ctrl + Shift + F4 关闭活动选项卡

Alt+Delete 安全删除

Ctrl + Alt + J 当前行使用模版

Ctrl +Ｊ插入模版

Ctrl+K 提交代码到 SVN

Ctrl+T 更新代码

Ctrl + Alt + Y 同步

Ctrl + ~ 快速切换方案（界面外观、代码风格、快捷键映射等菜单）

Shift + F12 还原默认布局

Ctrl + Shift + F12 隐藏/恢复所有窗口

Ctrl + Tab 转到下一个拆分器

Ctrl + Shift + Tab 转到上一个拆分器

三、调试部分、编译

Ctrl+F2 停止

Ctrl+F8 切换行断点

Ctrl+F9 生成项目

Alt+Shift+F9 选择 Debug

Alt+Shift+F10 选择 Run

Alt+Shift+F7 强制步入

Alt+Shift+F8 强制步出

Ctrl+Shift+F9 编译

Ctrl+Shift+F10 运行

Ctrl+Shift+F8 查看断点

Ctrl+Alt+F9 强制运行光标处

F7 步入

F8 步出

F9 恢复程序

Shift + F10 运行

Shift + F9 调试

Shift+F7 智能步入

Shift+F8 智能步出

Alt+F9 运行至光标处

Alt+F10 定位到断点

Alt+1 项目

Alt+2 收藏

Alt+6 TODO

Alt+7 结构
