1. 根据python版本下载对应的cx_Oracle版本

  • cx_Oracle下载： https://pypi.org/project/cx-Oracle/
  • pip install cx-Oracle

2. cx_Oracle和instantclient版本对应

  • instantclient下载： https://www.oracle.com/database/technologies/instant-client/downloads.html
  • 添加instantclient到环境变量（路径下包括oci.dll、oraocci11.dll、oraociei11.dll等）
  • 复制oci、oraocci11、oraociei11的3个DLL粘贴到PY目录的Libs/site-packages文件夹下面。
